```java
1.alt + insert: 创建类/包/接口
2.ctrl + alt + L: 格式化
3.ctrl + /: 单行注释
4.ctrl + shift + /: 多行注释
5.shift + 回车: 向下换行
6.ctrl + alt + 回车: 向上换行
7.ctrl + shift + 回车: 自动补全代码
8.ctrl + d: 复制
10.alt + shift + ↑/↓: 移动当前行
11.shift+f6: 批量修改相同名称的变量
12.alt+enter: 自动分配变量,创建类
13.ctrl+r: 替换
14.自动生成输出语句输出该变量
	变量.sout: 
15.自动生成变量:
            new ....().var
            new ....(): 写完后 alt + 回车
16.自动生成for循环
	数字.fori     生成从小到大的for循环
	数字.forr     生成从大到小的for循环
```
